Box Game

1. This game lasts for 2 minutes and has 5 levels.

2. A grid of 16 cells will be seen.

3. The cells will be highlighted in Green in a random order.

4. You need to memorize the order in which the cells are highlighted.

5. When the sequence ends, tap the cells in the same order they were highlighted.

6. The game will progress to the next level if you tap all the cells in the correct sequence.

7. If you fail in an attempt, the same level will be repeated. If you succeed in the second attempt, the game will proceed to the next level. However, if you fail, the game will exit.

8. Note that at higher levels, more cells will be highlighted in the sequence.
