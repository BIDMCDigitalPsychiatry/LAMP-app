Visual Association Task

1. This game lasts for 1 minute.

2. You will be shown 5 images consecutively. Each image will contain 2 objects mainly.

3. Next, you need to participate in a simple activity such as, answering a survey.

4. After the activity, the images that were shown previously will be shown.

5. However, the images will be incomplete.

6. The missing object will be shown with some others in a grid below the image.

7. Identify the missing element and tap it.

8. Do this for all the images in the sequence.
