Digit Span Forward

1. This game lasts for 3 minutes and has 6 levels.

2. The app will call out a number.

3. On the grid shown below the Speaker icon, you are required to tap the digits in the reverse order they are in the number.

  • a. For example, if the number 483 is called out, you need to tap 384 on the grid.

4. At higher levels, the digits in the number will increase.

5. The game will progress to the next level if you tap the cells correctly.

6. If you fail in an attempt, the same level will be repeated. If you succeed in the second attempt, the game will proceed to the next level. However, if you fail, the game will exit.
