Ti.Healthkit.SampleQuery

Description

HealthKit uses sample queries to search for sample data in the HealthKit store. You can use sample queries to search for any concrete subclasses of the Ti.Healthkit.Sample class, including Ti.Healthkit.CategorySample, Ti.Healthkit.Correlation, Ti.Healthkit.QuantitySample, and Ti.Healthkit.Workout objects.

Properties

limit : Number

The maximum number of samples that this query returns. (read-only)
