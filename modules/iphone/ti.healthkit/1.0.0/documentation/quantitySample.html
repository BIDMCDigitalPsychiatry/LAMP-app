Ti.Healthkit.QuantitySample

Description

The Ti.Healthkit.QuantitySample class is a subclass of Ti.Healthkit.Sample.

HealthKit uses quantity samples to represent data using a single numeric value. HealthKit provides a wide range of quantity types, letting you track a number of different health and fitness features.

Quantity samples are immutable: The sample's properties are set when the object is first created, and they cannot change.

Properties

quantity : Ti.Healthkit.Quantity

The quantity for this sample. (read-only)

aggregationStyle : Number

The aggregation style for the given quantity sample. See the QUANTITY_AGGREGATION_STYLE_XXX constants for possible values.
