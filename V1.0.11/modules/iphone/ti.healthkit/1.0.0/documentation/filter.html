Ti.Healthkit.Filter

Ti.Healthkit.Filter objects allow you to limit the number of objects returned by queries. Use the createFilterForObjects, createFilterForObjectsWithNoCorrelation, createFilterForSamples, createFilterForQuantitySamples, createFilterForCategorySamples, or createFilterForWorkouts function to create filter.

Ti.Healthkit.Filter objects do not have any functions or properties of their own.
