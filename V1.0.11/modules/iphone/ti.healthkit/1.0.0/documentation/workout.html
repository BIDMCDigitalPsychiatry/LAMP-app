Ti.Healthkit.Workout

Description

The Ti.Healthkit.Workout class a subclass of Ti.Healthkit.Sample.

HealthKit uses workouts to track a wide range of activities. The workout object not only stores summary information about the activity (for example, duration, total distance, and total energy burned), it also acts as a container for other samples. You can associate any number of samples with a workout. In this way, you can add detailed information relevant to the workout.

Workouts are mostly immutable. Their properties are set when the workouts are instantiated, and they cannot change. However, you can continue to add samples to the workouts.

Properties

duration : Number

The workout's duration in seconds. (read-only)

totalDistance : Ti.Healthkit.Quantity

The total distance traveled during the workout. (read-only)

totalEnergyBurned : Ti.Healthkit.Quantity

The total energy burned during the workout. (read-only)

activityType : Number

The type of activity performed during the workout. (read-only)

See the WORKOUT_ACTIVITY_TYPE_* constants for possible values.

workoutEvents : Array of Ti.Healthkit.WorkoutEvent objects

An array of workout event objects. (read-only)

This array specifies when the user has paused and resumed the workout activity.
