Ti.Healthkit.StatisticsQuery

Description

Statistics queries perform statistical calculations over the set of matching quantity samples. Statistics queries can calculate the minimum, maximum, and average value for discrete quantities. They can also calculate the sum for cumulative quantities.

You can create a StatisticsQuery object using the createStatisticsQuery function.

By default, these queries automatically merge the data from all of your data sources before performing the calculations. If you want to merge the data yourself, you can set the STATISTICS_OPTION_SEPARATE_BY_SOURCE option when you create the query. You can then request the statistical data for each source separately.

To calculate the statistics over a series of time intervals, use a Ti.Healthkit.StatisticsCollectionQuery.
